Snow 2000

demo

Just a classic snow animation made with JS, no fancy code, no breathtaking effect, a bit laggy and stuttering, but that's how it's supposed to be. Made by sedes.dev

Usage

No, it's not an npm package, so just copy this JS code snippet and paste it somewhere into your "application".


      
To run it, you just need to call _snow_2000 function (yeah, it's in global window object):

      
_snow_2000();

      
But ther's more! You can pass some options to it!

      
_snow_2000({
  className: 'some-class-name', // main wrapper class name, default: 'snow-2000'
  zIndex: 10 // z-index of main wrapper, not set by default
});
